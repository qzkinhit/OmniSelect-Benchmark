## 命题5(投票合成的互补优势:充分条件版)

**设定。** 取验证增益最高的 s 个候选,选集 S₁,…,S_s,|Sᵢ| = k。权重
wᵢ = max(ĝ(Sᵢ) − ĝ(random), 0),**前提:至少两个权重严格为正,否则投票合成退化为
跟随最优单一候选,命题不适用**。投票分 v(x) = Σ_{i: x∈Sᵢ} wᵢ,投票合成取 v 最大的
k 条(并列以真实性通道分数破平)。共同核 C = {x: 至少被两个候选选中},w↓ 为权重降序。

**条件。** (i) 机制互补:每条有害记录至多被一个候选选中;(ii) 共同核充足:|C| ≥ k;
(iii) 权重均衡:w_{(s-1)} + w_{(s)} > w_{(1)}。

**陈述。** 条件 (i)(ii)(iii) 下,投票合成的 top-k 选集 V ⊆ C,且 V ∩ H = ∅,即有害率
为 0。进一步设最优单一候选 S₁ 的有害占比 ρ(S₁) > 0,则由假设A(定理5)的一阶响应,
投票合成把有害项的超额风险降低 κ·ρ(S₁);当覆盖增量受控,即 η·(d(V) − d(S₁)) <
κ·ρ(S₁) 时,下游效用严格超过最优单一成员,否则保证限于有害率不升。

**证明。** 由 (i),任一 x ∈ H 的票数 ≤ w_{(1)},且 x ∉ C。C 中记录票数 ≥
w_{(s-1)} + w_{(s)} > w_{(1)}(条件 (iii)),故按票数排序时 C 全部位于 H 之前;由 (ii)
|C| ≥ k 得 V ⊆ C,而 C ∩ H = ∅,故 V ∩ H = ∅。下游结论由假设A 的
R = R₀ + κρ + ηd 两项分别代入。∎

**条件破坏时的界(非优雅退化,如实声明)。** 记 H₂ = {x ∈ H: 被至少两个候选选中}
(共识错误)。一般地 |V ∩ H|/k ≤ min(1, |H₂|/k + 1/k·|{x∈H: v(x) > min_{y∈V} v(y)}|),
且该界可达:一旦 (i) 破坏,共识错误恰被投票机制**排到最前**,退化不是渐进的。因此
互补程度必须实测:本文在图像臂实测 γ̂ = |H₂|/|H ∩ ∪Sᵢ| = 8.9%
(`experiments/vote_overlap_diag.log`),条件 (ii)(iii) 在该实例不满足(权重被单一
候选主导),投票合成如命题预测退化为跟随最优成员,有害率与增益与其持平,零损失。
在 TEP 与 ETTh1 上验证采纳投票合成并提升主结果,与"条件满足时严格改进"一致。
本命题是**充分条件**,不宣称解释全部实证采纳。

## 命题6(验证污染的有界影响;经收紧后强化)

**污染模型(显式)。** 验证集先 i.i.d. 抽取 n_v 条;随后一个**可见样本取值与候选组合**
的对手修改其中至多 ⌊ρ_v·n_v⌋ 条(标签、特征任意改,逐样本得分仍在 [0,1])。控制器
照常运行,不知道哪些被改、也不需要知道 ρ_v(描述性结论)。

**陈述。** 以概率 ≥ 1−δ,对全部 m 个候选同时有
|ĝ(π) − g(π)| ≤ ρ_v + ε(n_v, m, δ)。
由此分两层:
(1) **描述性结论(无需改算法、无需已知 ρ_v)**:偏差界与 argmax 择优照常成立,把定理4
中的 ε 替换为 ε + ρ_v;特别地,当最优与次优候选的真实验证间隙 Δ_val > 2(ε + ρ_v) 时,
控制器的选择与干净验证下完全相同。
(2) **规范性结论(门控保证)**:切换阈值 τ = 2ε 不变时,切换发生仅保证
G(σ_f) ≥ G(σ_r) − 2ρ_v,而真实优势 ≥ 4ε + 2ρ_v 的切换必被触发;若已知污染率上界
ρ̄_v 并把阈值抬为 τ = 2(ε + ρ̄_v),定理4 的无害性与灵敏性原样恢复。
推论D 的安全网相应退化为 2(ε + ρ_v)。

**证明(反事实全样本路线,覆盖自适应对手)。** 设 ĝ_full 为未被修改的反事实全样本均值。
每修改一条至多移动均值 1/n_v,共 ⌊ρ_v n_v⌋ 条,故 |ĝ − ĝ_full| ≤ ρ_v。对 ĝ_full
(真 i.i.d.)用 Hoeffding 与并集界得 |ĝ_full − g| ≤ ε(n_v, m, δ)。三角不等式即得。
该路线不依赖"留存子样本仍 i.i.d."(自适应对手下该假设不成立),故界对自适应、
候选感知的污染同样有效。∎

**含义(对抗口径,如实)。** 污染对**成对比较**的对抗摆动上限为 2ρ_v:对手可把两个候选
的估计差精准移动 2ρ_v,常数不可改进。本文实测验证间隙 0.02~0.15,故对抗安全污染率
上限约为 Δ_val/2 − ε(图像臂约 7% 量级)。实验网格 ρ_v ∈ {0.1, 0.2, 0.3} 检验的是
**随机污染**(实践中最常见的形态),其影响远小于对抗上界,不作为对抗界的证据。
ρ_v 接近 1/2 时安全区为空,超出本命题范围。

## 定理9'(跨模态预算分配:实测增益响应曲线上的水线;重述版)

**分配对象(重述)。** 不再假设可分离模型 κ_m·g_m(b)。直接以**实测留出增益响应曲线**
Γ_m(b) = ĝ_m(控制器在预算 b) − ĝ_m(随机在预算 b) 为分配对象:κ、有害占比变化与
覆盖项被曲线自然吸收,这正是分配应当使用的可实现边际收益。跨模态可加性由归一化保证:
各模态增益标准化到 [0,1](随机基线为 0,全量参考或 oracle 为 1),或由平台外生权重
ω_m 显式声明。预算 b_m 以绝对成本计(条数×单条成本),Σ_m b_m = B。

**陈述。** (1) 在实测曲线的凹区间(二阶差分 ≤ 0,可直接检验)上,归一化目标
Σ_m ω_m Γ_m(b_m) 的最优分配由水线刻画:κ 边际均衡 ω_m Γ_m′(b_m*) = λ,响应弱的模态
(如鲁棒基模,Γ_m ≈ 0)钉在下限 b_min。凸区间不在本陈述内,该处最优可在角点,用
贪心逐格点比较。(2) 自身比较静力学:∂b_m*/∂(响应强度) ≥ 0 对每个模态自身成立;
**跨模态排序**(响应大者分得多)仅在各模态响应形状同族(Γ_m′ 逐点可比)时成立,
一般不成立——高响应但快饱和的模态可以最优地分得更少,这一反例情形如实纳入陈述。
(3) plug-in 损失(三明治论证):以估计曲线 Γ̂_m(逐点误差 ≤ δ_m,由验证集中界给出,
格点插值误差另计 δ_m′)求解水线得 b̂,则
Σ ω_m Γ_m(b_m*) − Σ ω_m Γ_m(b̂_m) ≤ 2·Σ_m ω_m(δ_m + δ_m′)。
只需 Γ 有界,无 Lipschitz 反函数假设,κ → 0 不爆炸。

**证明要点。** (1) 凹区间上目标凹、约束线性,KKT 即水线。(2) 单调性:固定其他模态,
自身响应上移使一阶条件右移;跨模态反例:Γ₁ = 2(1−e^{-5b}),Γ₂ = 1·b,B = 2 时
水线给 b₁* < b₂* 尽管模态 1 响应更强。(3) 三明治:F(b*) − F(b̂) ≤ [F − F̂](b*) +
[F̂(b*) − F̂(b̂)] + [F̂ − F](b̂),中项 ≤ 0(b̂ 是 F̂ 的最优),两端各 ≤ Σω(δ+δ′)。∎

**与既有工作的关系。** RegMix 与 DoReMi 在文本域之间学混比,依赖代理模型训练;本定理
在模态之间按实测响应分配预算,权重来自控制器本就输出的留出增益,无需额外训练。

**入稿门槛(裁决要求,如实执行)。** 本定理在预算分配实验(实测视觉与 TEP 的响应曲线,
检验凹性、比较等分与响应加权分配)验证通过之前**不进正文**,只以当前重述版放算力扩展
或附录;实验通过后按实测曲线的水线分配入方法章。
